wiki-elements

wiki-profile

The wiki-profile element is a WebComponent to display the profile image of a person by giving the name.

Attributes

  • language: The language code of the article. Default is en.
  • article: The title of the article.

Examples


                    <wiki-profile language="en" article="Barack Obama">
                    </wiki-profile>
                

                    <wiki-profile language="en" article="Albert Einstein">
                    </wiki-profile>
                

                    <wiki-profile language="en" article="Ethan Hawk">
                    </wiki-profile>
                    <wiki-profile language="en" article="Julia Roberts">
                    </wiki-profile>
                    <wiki-profile language="en" article="Mahershala Ali">
                    </wiki-profile>
