wiki-elements

wiki-video

The wiki-video element is a custom element that displays a video from Wikimedia Commons.

Video caption and attribution to the author of the image is automatically added.

Attributes

  • source: The source of the video. It can be a URL or a file name from Wikimedia Commons.

Examples


                      <wiki-video source="File:Train_Crimean_Bridge_2019.webm">
                    </wiki-video>
